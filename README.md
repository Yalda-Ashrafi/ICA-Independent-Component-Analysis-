## ICA-Independent-Component-Analysis-
Survey Data Analysis using ICA

This project explores survey data preprocessing and dimensionality reduction using Independent Component Analysis (ICA) in Python.

 Project Overview

The dataset contains:

10,716 survey responses
355 columns/features

The analysis was performed in Google Colab using Python libraries such as Pandas and Scikit-learn.

Technologies Used
Python
Google Colab
Pandas
NumPy
Scikit-learn
Project Steps
Data loading and exploration
Handling missing values
Converting categorical data into numeric format
Data scaling and preprocessing
Applying ICA (Independent Component Analysis)
Extracting important components from the dataset

Kaggagle dataset:
https://www.kaggle.com/datasets/kaushal200/surveydata

📊 Machine Learning Technique

This project uses FastICA for dimensionality reduction and feature extraction.

 Learning Outcome

Through this project, I learned:

Data preprocessing techniques

Feature transformation
Dimensionality reduction using ICA
Data analysis workflows in Python
